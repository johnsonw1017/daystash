## Daystash

Daystash is a personal journaling web app for capturing daily moments through text and photos, designed as a private travel log and memory archive.

Development in progress...

## Testing

Run the Vitest test suite:

```bash
npx vitest
```

Run tests once, without watch mode:

```bash
npx vitest run
```

Open the Vitest browser UI:

```bash
npx vitest --ui
```
